Tentang Color

HTML mendukung beberapa format penulisan warna, tidak hanya menggunakan nama warna seperti red, blue, atau green, tetapi juga format lain seperti HEX, RGB, dan HSL.

Format-format ini memberikan akurasi lebih dalam menentukan warna dibandingkan hanya dengan nama warna.

Color Picker:

Salah satu situs yang dapat digunakan untuk memilih warna adalah: https://colorpicker.me.

Di sana, kita dapat memilih warna yang diinginkan dan menyalin kode HEX untuk diterapkan di HTML.



Pembahasan List

1. Unordered List

Untuk membuat daftar tanpa urutan (Unordered List), kita bisa menggunakan tag <ul> di HTML.

Setiap item dalam daftar tersebut menggunakan tag <li>.


2. Ordered List

Untuk membuat daftar berurutan (Ordered List), kita bisa mengganti tag <ul> menjadi <ol>, dan itemnya tetap menggunakan tag <li>.

Ordered List Type:

Pada daftar yang berurutan (<ol>), kita bisa mengubah jenis penomorannya dengan menambahkan atribut type pada tag <ol>. Format yang bisa digunakan antara lain:

1. type="1": Menggunakan angka (default).
2. type="A": Menggunakan huruf kapital.
3. type="a": Menggunakan huruf kecil.
4. type="I": Menggunakan angka Romawi kapital.
5. type="i": Menggunakan angka Romawi kecil.
